```
1. 人们其实不知道：最大的快乐是内在寻求达到完全控制念头的境界，你不是念头，你观察念头，利用好的念头，断掉不好的念头，不需要时就不要有任何念头阻止你感受当下。
2. 好的戒色状态是非常宝贵的，一旦破戒就很难恢复到这个戒色状态。
```

##### 我的困惑：

```
1. 虽然已经控制住了各种各样的念头，但是我仍然感觉不太好，这种感觉不是由念头照成的
2. 不太好的原因：
	1. 我总是观察我的身材是否有变好
	2. 观察我的精神状态是否有变好
	3. 没有价值输出照成的心理负担或者说缺失
```
